# 노션에서 후기 작성하는 방법

이 프로젝트는 별도의 로그인/글쓰기 화면이 없습니다. 모든 후기는 **노션 데이터베이스에서 직접 작성**하며, 웹사이트는 "공개" 상태인 후기만 자동으로 보여줍니다.

## 📋 데이터베이스 속성 (한 번만 설정)

Notion에 새 데이터베이스를 만들고, 아래 속성명·타입을 **정확히 그대로** 만들어주세요. 속성명이 코드와 1:1로 매칭되므로 오타가 있으면 안 됩니다.

| 속성명        | 속성 타입     | 설명                                |
| ------------- | ------------- | ----------------------------------- |
| **후기 제목** | Title         | 예: "거문도 참돔 조황 대박"         |
| **작성자**    | Rich text     | 표시용 닉네임                       |
| **사진**      | Files & media | 여러 장 업로드 (드래그로 순서 지정) |
| **후기 내용** | Rich text     | 본문                                |
| **공개 여부** | Select        | 옵션: "공개" / "비공개(작성중)"     |

작성일시는 별도 속성을 만들 필요 없이 Notion의 자동 "생성 일시"를 그대로 사용합니다.

## ✍️ 새 후기 작성하기

1. 데이터베이스에서 **새 페이지** 추가
2. 위 표의 속성을 채워 넣기 (사진은 여러 장 업로드 가능)
3. 작성이 끝나기 전까지 **"공개 여부"는 반드시 "비공개(작성중)"**으로 둘 것
4. 다 작성했으면 **"공개 여부"를 "공개"로 변경** → 웹사이트에 즉시(최대 60초 이내) 노출됨

## 🔌 최초 1회 설정

1. [Notion Integrations](https://www.notion.so/my-integrations)에서 새 Integration 생성 후 토큰 복사
2. 데이터베이스 우측 상단 "..." → "Add connections" → 생성한 Integration 연결
3. 데이터베이스 URL에서 32자리 Database ID 복사
4. `.env.local`에 `NOTION_API_KEY`, `NOTION_DATABASE_ID` 값으로 입력

## ⚠️ 주의사항

- **속성명을 나중에 바꾸면 코드도 함께 고쳐야 합니다.** 처음 정한 이름을 그대로 유지하는 것을 권장합니다.
- Notion에 업로드한 사진의 실제 파일 URL은 약 1시간 후 자동 만료되지만, 페이지를 새로고침하면 문제없이 다시 표시됩니다.
- 비공개(작성중) 상태인 후기는 목록/상세 페이지 어디에도 노출되지 않습니다.
